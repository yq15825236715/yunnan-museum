# 云南古建筑数字馆

## 项目简介

云南古建筑数字馆是一个展示云南省63座古建筑的全景式数字博物馆，涵盖13个州市、15个民族、650年历史跨度。

## 快速启动

### 方式一：双击启动（推荐）

双击运行 `快速启动.bat`，选择启动方式：
- 选项1：Python HTTP服务器（推荐，功能完整）
- 选项2：直接打开文件（最快，部分功能受限）
- 选项3：Node.js服务器（需要安装Node.js）

### 方式二：命令行启动

```bash
# Python
python -m http.server 8080

# Node.js
npx http-server -p 8080

# 然后访问 http://localhost:8080
```

## 展厅功能

| 展厅 | 功能 | 可视化 |
|------|------|--------|
| 时光长廊 | 按年代浏览建筑 | 折线图、柱状图、时间轴 |
| 地理长卷 | 按地区浏览建筑 | 交互地图、筛选器 |
| 建筑万象 | 按类型浏览建筑 | 饼图、雷达图、柱状图 |
| 民族交融 | 按民族浏览建筑 | 散点图、民族图标 |
| 匠心传承 | 建筑技术展示 | 技术卡片、表格 |
| 守护重生 | 保护与重建 | 饼图、柱状图、散点图 |
| 回望·尾声 | 数据总结 | 问答式展示 |
| 建筑图鉴 | 建筑详情 | 3D模型、珐琅图、筛选搜索 |

## 技术依赖

### 本地依赖（无需网络）
- ECharts 图表库（js/echarts.min.js）
- Model Viewer 3D查看器（js/model-viewer.min.js）
- 云南省地图数据（assets/data/）
- 所有图片、模型资源

### 在线依赖（需要网络）
- 高德地图API（hall-map.html 地理长卷展厅）
  - 如无网络，地图功能不可用，但其他功能正常
